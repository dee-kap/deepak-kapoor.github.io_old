Download Oslo Tools and SDK January 2009 CTP

Feb 3rd, 2009 | Comments

Microsoft has released January 2009 CTP of Oslo SDK and Tools.

You can download the CTP here.

The SDK includes Intellipad tool and "M" code samples to get you started. This release also support Visual Studio 2008. You can find more details about January 2009 CTP at SellsBrothers.com.

Here are few links to get you started with Oslo:

  1. What is Oslo? By Douglas Purdy
  2. Oslo Developer Center on MSDN
  3. Oslo Forums on MSDN
  4. The Origins of Microsoft’s Oslo Software Modeling Platform

 

I am sure that there is someone out there writing a book on Oslo. At present there is only Draft Specification available from Amazon.
