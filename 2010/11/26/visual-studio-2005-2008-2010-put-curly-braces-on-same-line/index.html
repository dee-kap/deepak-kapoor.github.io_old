Visual Studio 2005 2008 2010 Put Curly Braces on Same Line

Nov 26th, 2010

I have been looking at some code from Microsoft and I found that in C# they keep curly braces on the same line. This is also the preferred style in Java. Since I also write Java code these days I decided to keep the same style for curly braces. So how do you configure Visual Studio to place curly braces on the same line? It turns out that there are options which allow you to do this. In Visual Studio go to Tools –> Options –> Text Editor –> C# –> Formatting and toggle check boxes under New line options for braces. This will not affect  existing code but any new code you write will honour these new settings.
