Anyhow, the way to fix this issue is by adding guava-r09-jarjar.jar to build path. In Netbeans add the jar to Libraries for your project.

guava-r09-jarjar.jar can be found in /usr/lib/hadoop-0.20/lib

or use the locate command to find it on your system

locate guava-r09-jarjar.jar
