IBM BigInsights Web Console Issue With BigDataUniversity VM

Jun 30th, 2012

BigInsights web console error in IBM Infosphere BigInsights VM usually occurs because web console is not running. It can be started by executing start.sh which can be found in /opt/ibm/biginsights/bin. I discovered this issue while working through lab 3 of Hadoop Fundamentals 1 course from BigDataUniversity.

Steps to resolve the issue:

Assuming that you are logged in as root. Execute these commands on terminal.

cd /opt/ibm/biginsights/bin

./start.sh console
