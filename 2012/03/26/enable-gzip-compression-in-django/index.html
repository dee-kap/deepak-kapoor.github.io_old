Enable GZip Compression in Django

Mar 26th, 2012

GZip compression reduces size of payload sent to browser which in turn increases performance for your site. If you are not doing this then stop everything and read this. Hopefully you will be convinced. I for one use it all the time.

How do we do it in Django? Just include GZip Middleware. How? Easy. In your settings.py file look for MIDDLEWARE_CLASSES and add

django.middleware.gzip.GZipMiddleware

Here it is in my settings.py along with other middleware classes I am using.

MIDDLEWARE_CLASSES = (
    'django.middleware.gzip.GZipMiddleware',
    'django.middleware.common.CommonMiddleware',
    'django.contrib.sessions.middleware.SessionMiddleware',
    'django.middleware.csrf.CsrfViewMiddleware',
    'django.contrib.auth.middleware.AuthenticationMiddleware',
    'django.contrib.messages.middleware.MessageMiddleware',
)
