Free Icons and Images With Visual Studio 2008

Aug 6th, 2008

Visual Studio 2008 comes with an Image Library which contains thousands of icons, animations and images which you can use in your applications. You can find the Image Library at C:Program FilesMicrosoft Visual Studio 9.0Common7VS2008ImageLibrary1033 Just unzip VS2008ImageLibrary.zip file located at this location and you have most of the graphics used by Microsoft in many applications including Visual Studio itself.
