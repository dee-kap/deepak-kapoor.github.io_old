Windows Azure - Connect to Linux Virtual Machine

May 31st, 2013

In an earlier post I created a Linux virtual machine on Windows Azure. In this post I will show you how to establish a SSH connection to the virtual machine.  

Windows

Windows does not come with a SSH client but we can use PuTTY to make SSH connections from a Windows machine. Open putty.exe and enter host information as it is on the management portal.     Click Open and you are asked to login. Enter username and password and you are logged into the Linux virtual machine.    

Linux Terminal

Logging into Linux virtual machine on Windows Azure from a Linux terminal is dead simple. Linux already comes with a OpenSSH client so there is nothing to be installed. Just run the ssh command with correct port number and host. In my case the command is

ssh –p 58473 azureuser@debrelubuntu.cloudapp.net

  Enter the correct password when prompted and Bob’s your uncle.
